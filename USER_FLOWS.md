# User Flows

1. **Landing** → User reads value → Clicks "Start".
2. **Chat** → User types "We need a CRM" → AI asks "Who are your users?" → User replies → (repeat 4x).
3. **Generate** → User clicks "Generate Report" → Loading spinner appears → Report renders below.
4. **Refine** → User clicks on report text → Edits it manually (demonstrates human-in-the-loop).
5. **Export/Send** → Clicks "Send to Consultant" → Sees success state.
