# AI Business Discovery Assistant

## Overview
A prototype SaaS application demonstrating how AI accelerates software consultancy discovery phases.

## Core Value
Turns a raw customer conversation into:
- Business problems & pain points
- Functional requirements & user stories
- Technical architecture suggestions
- A dynamic project effort pie chart

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI API (GPT-4o-mini)
- Chart.js

## Deployment
- Vercel (ready to deploy)

- All specifications are in the root directory as .md files.
